# KMD Editor - Implementation Plan

## 1. 韵律标准与排版进化 (v1.1.0) - [DONE]
- [x] MD 语法解糖 (`**`, `*`, `#`, `---`)
- [x] Baseline V2 基线锚定排版
- [x] 相对字距 (0.02em Tracking)
- [x] 生产级 CJK 字体加载 (FontFace API)
- [x] 样式去重逻辑 (解决双重缩放 Bug)

## 2. Monaco 编辑器与智能增强 (v1.1.5) - [DONE]
### 2.1 高级 Monarch 语法高亮
- [x] **转义符支持**: 对 `\{`, `\>`, `\@` 等进行特殊着色。
- [x] **容器着色**: 为 `{...}` 和 `[...]` 内部的文字提供独特的背景或前景区分。
- [x] **指令精细化**: 区分 `@` 指令名、命名空间 (`cam.`, `f.`) 与参数。
- [x] **节奏糖高亮**: 对 `!`, `|`, `~`, `^`, `>` 等符号进行节奏感着色。
- [x] **头文件增强**: 真正的 YAML 风格头文件高亮。

### 2.2 IntelliSense 自动补全
- [x] **触发器定义**: 监听 `.`, `f.`, `cam.`, `@`, `:`, `(` 触发补全。
- [x] **上下文感知补全**:
  - `f.` -> 仅显示视觉特效 (Filters/Behaviors)。
  - `cam.` -> 仅显示镜头与舞台指令。
  - `:` -> 提供 `char`, `group`, `block` 层级后缀。
  - `(` -> 提供 `var.xxx` 变量与 `mark.xxx` 标记建议。

## 3. 编辑器集成与生态 (v1.1.5) - [DONE]
- [ ] **快捷键系统**: 实现 Ctrl+Enter (Run), Alt+N (Next) 的编辑器内部绑定。
- [x] **实时错误诊断**: 将 KMDParser 的验证结果（含未知指令校验）实时标注在 Monaco 边栏。
- [x] **中文友好**: 自动屏蔽中文全角符号（如 `：`）的歧义字符警告。
- [ ] **VS Code 插件**: 规划基于 v1.1.5 语法的实时高亮插件。

## 4. 视觉质感与逻辑后续
- [x] **持久化节奏系统**: 修复速度糖衣的累加效应，实现行级 (`~`/`^`) 与组级 (`:group`) 的自动重置逻辑。
- [ ] 西文 Kerning Pair 自动微调。
- [ ] 3D 转场或纹理溶解特效探索。
